# README
- This project is best suited for a "Pixel 3a", and the UIs will look best on this device

## How it works
- The user will be prompted to choose one of the options available on the Main Activity
- This will then collect a series of data based on the Pokemon available in that game/group of games from an online API (pokeapi.co)
- Once the data is collected, which will take a small period of time due to the volume of data collected, the user will be redirected to an area where the user will be able to enter the name of a Pokemon
- Entering the name of a Pokemon from the selected game(s) will show the selected Pokemon's image in the display, as well as it's type(s) below the name.
- If the name of a Pokemon not from the selected game(s), or any other erroneous response, a "Toast" message will appear notifying the user of their error
